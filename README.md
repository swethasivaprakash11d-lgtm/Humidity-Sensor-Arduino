# Humidity Sensor using Arduino UNO

## Overview

This project demonstrates the interfacing of a humidity sensor with Arduino UNO to measure ambient humidity in real time. The Arduino continuously reads the humidity values from the sensor and displays the readings through the Serial Monitor or an LCD display.

## Components Used

* Arduino UNO
* Humidity Sensor (DHT11/DHT22 or equivalent)
* Breadboard
* Jumper Wires
* USB Cable

## Working Principle

The humidity sensor measures the moisture content in the surrounding air and sends the data to the Arduino. The Arduino processes the received values and displays the humidity percentage in real time. This enables continuous monitoring of environmental conditions.
